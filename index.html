CSS Topic: Box Shadow

box-shadow

The box-shadow property in CSS adds a shadow effect around an element's frame. Box shadows are described by X and Y offsets relative to the blur, radius, and color.

See the Pen Box Shadow by Nicole Parker (@naparker) on CodePen.

You can have multiple shadows by separating the specifications with a comma.

See the Pen multiple shadows by Nicole Parker (@naparker) on CodePen.

inset

Box shadows can also be inside the text box by using the inset value inside of the box-shadow.

See the Pen inset by Nicole Parker (@naparker) on CodePen.

Zero offset and blur

Having the offset and blur set to zero makes the box shadow a solid-colored outline of equal size on all sides. This can be used instead of outline

See the Pen zero offset by Nicole Parker (@naparker) on CodePen.

When we put all the features of box-shadow together, it can create a really cool effect!
